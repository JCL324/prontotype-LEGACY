Using Macros

Macros are little helpers that you can easily define once and then use throughout your templates to make sure that common HTML snippets stay editable in one place. They are a bit like page includes, except that they can use optional parameters to adjust the HTML or content output on a per-use basis, which makes them very flexible.

Built in Macros

Prontotype also comes with some built-in macros to make it quick to prototype common things like forms, navigation and more.

And don't forget, if the markup that these macros output isn't quite to your taste you can easily override them by defining your own macros at any time. See the documentation on macros for more information.

Example: A simple registration form

{{ form.form_open }} {{ form.input('User name') }} {{ form.input('Password', 'password') }} {{ form.textarea('Tell us a little about yourself') }} {{ form.checkbox('I agree to the terms and conditions') }} {{ form.button('Register') }} {{ form.form_close }}
