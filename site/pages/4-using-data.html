Using data in your Prototype

Prontotype makes it really easy to define data and then use it throughout your prototype. If you create a new YAML or CSV file in the /data directory then the data contained in it will then be available in your templates for you to use.

So, for example, if you have a file called /data/posts.yml which defined a number of posts that each have a title, you could do something like this to display the title of the first one:

{% raw %}{{ data.posts[0].title }}{% endraw %}

For more information on data formatting and using template tags to access your data from within your pages you should check out the full documentation.
